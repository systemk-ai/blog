物体検出モデルを ImageNet で事前学習し、社内データセットで微調整した。本稿では、少数データ
条件下での精度改善と、評価設計について述べる。

## 実験設定

学習データは社内で収集した 12,000 フレーム。評価は held-out の 2,000 フレームで行い、指標は
mAP@0.5 を用いる。平均適合率（AP）は次式で与えられる。

$$
\mathrm{AP} = \int_0^1 p(r)\,dr
$$

ここで $p(r)$ は再現率 $r$ における適合率である。

## 学習ループ

簡略化した学習ループを示す。

```python
for images, targets in loader:
    loss = model(images, targets)
    loss.backward()
    optimizer.step()
    optimizer.zero_grad()
```
